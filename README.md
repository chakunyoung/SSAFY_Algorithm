# SSAFY_Algorithm
문제안풀면 오마카세
<br><br>

## 진행일정
- 주 1회 (수요일) - 7시부터 9시
- 장소 : 강남역 근처 스터디카페
<br><br>

## 진행방법
- (일단) 알고리즘 정규과정 들어가기 전까지 구글시트에 있는 문제 다 풀기 (~2월 8일)
- 구글시트 문제 끝나면 상1중2 또는 상1중1하1 문제 정해서 풀기
    - 문제마다 마크다운 문서로 생각의 흐름 적어보기 (피드백)
- 과제 안 해오면 전체 스터디원 오마카세 쏘기 (1주일 유예기간 有)
- 공통 문제 정해서 각자 풀어오고 1시간동안 코드 리뷰
- 구현 위주로 진행
